### Add Cloud SQL Instance Connection to Deployment

Added Cloud SQL instance connection configuration to deployment workflow.

**What's New:**
- Added Cloud SQL instance connection to Cloud Run deployment
- Enables secure connection to managed PostgreSQL database

**Why This Matters:**
Cloud SQL provides managed PostgreSQL for production workloads. This configuration enables the deployed application to securely connect to the production database through Cloud SQL's private network.

**Technical Details:**
- Updated `.github/workflows/deploy.yml`
- Added Cloud SQL instance connection configuration
- Enables secure database connectivity in production
