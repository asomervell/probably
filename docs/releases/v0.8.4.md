# Release v0.8.4

## 🧹 Cleanup

### Update Deploy Workflow to Match GitHub Secrets/Variables

Cleaned up deployment workflow by removing references to deleted secrets and variables.

**What's Changed:**
- Removed references to deleted secrets: `CLOUD_SQL_CONNECTION_NAME`, `GCS_CREDENTIALS_JSON`
- Removed references to deleted variables: `CATEGORIZATION_LOG_FILE`, `INSIGHTS_LOG_FILE`, `SYNC_LOG_FILE`
- Removed `--add-cloudsql-instances` flag since `CLOUD_SQL_CONNECTION_NAME` was removed

**Why This Matters:**
Keeping the deployment workflow in sync with actual GitHub secrets and variables prevents deployment failures and reduces confusion. Removing unused configuration makes the workflow easier to understand and maintain.

**Technical Details:**
- Updated `.github/workflows/deploy.yml`
- Removed obsolete secret and variable references
- Simplified Cloud SQL connection configuration
