# Release v0.8.15

## 🐛 Bug Fixes

### Fixed GCloud Authentication Order

Fixed the authentication order in the deployment workflow to ensure gcloud is properly authenticated before setup.

**What Was Fixed:**
- **Correct authentication sequence** - Moved `Authenticate to Google Cloud` step before `Set up Cloud SDK`
- **Proper credential flow** - Ensures service account credentials are available before gcloud setup

**Why This Matters:**
The Cloud SDK setup needs authentication credentials to be available first. Reversing the order caused authentication failures in the deployment pipeline.

**Technical Details:**
- Swapped the order of `google-github-actions/auth@v2` and `google-github-actions/setup-gcloud@v2` steps
- Ensures credentials are loaded before gcloud initialization
