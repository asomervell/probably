### Added DATABASE_URL Debugging in Deployment

Added debug output to help diagnose database connection issues during deployment.

**What's New:**
- **Database URL validation** - Outputs DATABASE_URL length and first 20 characters (masked for security)
- **Deployment debugging** - Helps identify if DATABASE_URL is missing or malformed

**Why This Matters:**
Database connection failures are common deployment issues. This debug output helps quickly identify if the problem is a missing or incorrectly configured DATABASE_URL secret.

**Technical Details:**
- Added debug step that outputs DATABASE_URL length and prefix
- Values are masked to prevent exposing sensitive connection strings
- Helps diagnose configuration issues before deployment fails
