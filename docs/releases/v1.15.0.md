# v1.15.0 - Akahu Integration for New Zealand Banks

**Release Date:** January 1, 2026

This release introduces full support for New Zealand bank connectivity through Akahu, expanding Probably's reach to international users.

## ✨ Features

### Akahu Integration for New Zealand Banks

Probably now supports connecting New Zealand bank accounts through [Akahu](https://www.akahu.nz/), New Zealand's open finance platform.

**Key Features:**
- **OAuth 2.0 Authentication**: Secure authorization flow with token exchange and automatic refresh
- **Multi-Bank Support**: Connect accounts from all major NZ banks supported by Akahu
- **Automatic Account Sync**: Accounts are automatically discovered and synced
- **Transaction Synchronization**: Full transaction import with NZD currency support
- **Real-time Webhooks**: `TOKEN:DELETE` and `ACCOUNT:UPDATE` events for instant updates

**Settings UI Enhancements:**
- Provider-aware interface with sync/disconnect buttons per provider
- Multi-provider dropdown when both Teller (US) and Akahu (NZ) are configured
- Visual provider badges (🇺🇸 US Banks, 🇳🇿 NZ Banks)

## 🔧 Technical Details

### New Components

- `internal/sync/akahu.go` - Akahu API client
- `internal/sync/akahu_provider.go` - Provider interface wrapper
- `internal/handlers/akahu.go` - OAuth and webhook handlers
- `internal/sync/akahu_test.go` - Unit tests

### Environment Variables

```bash
AKAHU_APP_ID=app_xxx
AKAHU_APP_SECRET=xxx
AKAHU_ENVIRONMENT=sandbox  # or production
AKAHU_WEBHOOK_SECRET=xxx
```

### API Endpoints

- `GET /connections/connect/akahu` - OAuth connection flow
- `GET /connections/callback/akahu` - OAuth callback
- `POST /connections/sync/akahu/{connectionId}` - Manual sync
- `DELETE /connections/disconnect/akahu/{connectionId}` - Disconnect
- `POST /api/akahu/webhook` - Webhook handler
