# v1.30.0 - Passkey Authentication

**Released:** January 3, 2026

## Overview

This release introduces passkey authentication, providing a modern, secure, and passwordless login experience. Passkeys leverage WebAuthn technology to enable authentication using your device's built-in biometric authentication (Face ID, Touch ID, Windows Hello) or hardware security keys.

## Features

### Passkey Authentication
- **Passwordless Login**: Sign in using passkeys instead of passwords
- **Multiple Device Support**: Register passkeys on multiple devices (laptop, phone, tablet)
- **Discoverable Credentials**: Sign in without entering your email address
- **Biometric & Hardware Key Support**: Works with Face ID, Touch ID, Windows Hello, Android biometrics, and hardware security keys

### Security Settings Page
- New `/settings/security` page for managing passkeys
- View all registered passkeys with device names and last used timestamps
- Register new passkeys for additional devices
- Remove passkeys you no longer use

### Login Page Updates
- New "Sign in with passkey" button
- Works alongside traditional password login
- Auto-detects browser passkey support

## Technical Details

### Database
- Migration 057: Added `passkeys` table for WebAuthn credentials

### New Dependencies
- `github.com/go-webauthn/webauthn` v0.15.0

### API Endpoints
- `POST /auth/passkey/login/begin` - Start passkey login
- `POST /auth/passkey/login/finish` - Complete passkey login
- `GET /auth/passkey/check` - Check if user has passkeys
- `POST /auth/passkey/register/begin` - Start registration
- `POST /auth/passkey/register/finish` - Complete registration
- `GET /settings/security` - Security settings page
- `DELETE /settings/security/passkeys/{id}` - Remove passkey

## Browser Compatibility

Passkeys supported in all modern browsers:
- Chrome/Edge 67+
- Firefox 60+
- Safari 13.1+ (macOS) / 14+ (iOS)
- Opera 54+
