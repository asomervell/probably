# Release v0.5.5

## ✨ Features

### Add Bud Enrichment Integration and Merchant Management

Added Bud API integration for transaction enrichment and merchant management.

**What's New:**
- **Bud API Client** - Integration with Bud API for transaction enrichment
- **Enrichment Features** - Logo fetching, merchant name resolution, category assignment
- **Enrichment Worker** - Background worker to process transactions
- **Merchant Management** - Improved merchant data handling

**Why This Matters:**
Bud API provides rich transaction enrichment data including merchant logos, standardized merchant names, and categories. This integration automates the process of enriching transactions with useful metadata.

**Technical Details:**
- Added Bud API client for transaction enrichment
- Created enrichment worker for background processing
- Enhanced merchant management system
- Automatic logo and category assignment
