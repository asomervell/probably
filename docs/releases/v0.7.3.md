### Shadcn Gomponents

Added shadcn/ui component library integration using gomponents.

**What's New:**
- Integrated shadcn/ui components using gomponents
- Added reusable UI components for consistent design
- Improved component library structure

**Why This Matters:**
shadcn/ui provides a comprehensive set of well-designed, accessible UI components. Using gomponents allows these components to be used in Go server-side rendering while maintaining consistency.

- Added shadcn component implementations
- Integrated with gomponents for Go templating
- Enhanced UI component library
