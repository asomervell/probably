# Release v0.8.2

## ✨ Features

### Store Only Filenames in Database, Construct CDN URLs at Display Time

Refactored logo storage to store only filenames in the database and construct CDN URLs dynamically.

**What's New:**
- Changed logo storage to return only filenames (not full URLs)
- Updated all display code to construct CDN URLs from filenames using `getLogoURL` helper
- Removed external URL storage - only store downloaded filenames
- Added migration script to extract filenames from existing URLs

**Why This Matters:**
Storing only filenames makes the system more flexible - CDN URLs can be changed without updating the database. This allows for easier CDN migration, different CDN configurations per environment, and simpler logo management.

**Technical Details:**
- Logo URLs are now constructed at display time using a helper function
- Database stores only the filename
- Migration script helps convert existing full URLs to filenames
- More flexible CDN configuration
