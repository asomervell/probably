# Release v0.3.5

## 🔧 Refactoring

### Flatten Project Structure

Major refactoring to simplify project structure by removing archive folder and moving Go code to root.

**What's Changed:**
- Deleted `archive/` folder (old Rails codebase)
- Moved `go/` contents to project root
- Updated all path references in scripts, docs, and configs
- Simplified install.sh and README for new structure

**Why This Matters:**
A flatter project structure is easier to navigate and maintain. Removing the archive folder and moving Go code to the root eliminates unnecessary nesting and makes the project structure clearer.
