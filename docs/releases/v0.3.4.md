# Release v0.3.4

## 🔒 Security

### Remove .secure from History, Add Teller Setup Script

Security improvement to remove sensitive certificates from git history and added setup script for Teller configuration.

**What's Changed:**
- Added `.secure/`, `*.pem` to .gitignore (certificates should never be committed)
- Removed certificate.pem and private_key.pem from git history
- Added interactive `setup-teller.sh` script for easy bank sync configuration
- Updated README with comprehensive Teller setup instructions

**Why This Matters:**
Sensitive certificate files should never be committed to git. This security fix removes them from history and ensures they won't be committed in the future. The setup script makes it easier to configure Teller bank syncing securely.

**Technical Details:**
- Updated .gitignore to exclude certificate files
- Removed sensitive files from git history
- Created interactive setup script for Teller configuration
- Enhanced documentation for secure setup
