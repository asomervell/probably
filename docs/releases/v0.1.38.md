# Release v0.1.38

## ✨ Features

### Backup and Restore Functionality

Added comprehensive backup and restore functionality allowing users to export and import their financial data.

**What's New:**
- **Backup export** - Export all ledger data (accounts, transactions, tags, rules, merchants) to a ZIP file
- **Backup import** - Import backup files to restore or migrate data
- **Intelligence page** - New intelligence/dashboard page for financial insights
- **Auth improvements** - Redirect to intelligence page after login/register
- **Handler updates** - Enhanced transaction, tag, and rule handlers

**Why This Matters:**
Backup functionality is essential for data portability and peace of mind. Users can now export their financial data and import it elsewhere, or use backups for disaster recovery.

**Technical Details:**
- Created `internal/backup/backup.go` (755 lines) with export/import logic
- Added comprehensive test suite (661 lines)
- Backup creates ZIP file with JSON files for each data type
- Import recreates all data in a new ledger
- Added backup handlers for download/upload
- Created intelligence page handler
