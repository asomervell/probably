## ✨ Features

### Add LLM Tool Calling Framework for AI Chat Assistant

Implemented a complete tool calling system for conversational finance management, enabling the AI chat assistant to interact with financial data through structured function calls.

**What's New:**
- Added comprehensive LLM tool calling framework
- Created executor system for running tool functions
- Implemented chat interface with tool support
- Added database migrations for entities and recurring patterns
- Enhanced logo client for better merchant identification
- Added handlers for entities and patterns management

**Why This Matters:**
The tool calling framework enables natural language interactions with financial data. Users can ask questions like "What did I spend on groceries last month?" and the AI can execute the appropriate database queries and return formatted results. This makes financial data exploration much more intuitive and accessible.

**Technical Details:**
- Created `internal/llm/executor.go`, `internal/llm/handler.go`, `internal/llm/prompts.go`, and `internal/llm/tools.go`
- Added database migrations: `033_detected_recurring.sql`, `034_subscription_price_history.sql`, `035_recurring_amount_bucket.sql`, `036_entities.sql`
- Enhanced `internal/enrichment/logo_client.go` for better logo extraction
- Updated `internal/handlers/accounts.go`, `internal/handlers/entities.go`, `internal/handlers/patterns.go`
- Added entity model in `internal/models/entity.go`
- Updated `internal/insights/aggregator.go` and `internal/insights/analyzer.go` for entity support
