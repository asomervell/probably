# Release v0.1.32

## ✨ Features

### Load .env File in Server Startup

Added automatic loading of `.env` file during server startup for local development.

**What's New:**
- Server now automatically loads `.env` file on startup
- Simplifies local development configuration

**Why This Matters:**
Loading `.env` files automatically makes local development easier by eliminating the need to manually source environment variables or use external tools.

**Technical Details:**
- Added `.env` file loading in `go/cmd/server/main.go`
- Uses `godotenv` or similar to load environment variables
- Only affects local development (production uses environment variables from Cloud Run)
