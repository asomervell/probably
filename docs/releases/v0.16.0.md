## ✨ Features

### Restructure Pages - Add Pulse Dashboard, Rename Intelligence to Statements

Major navigation restructure introducing the new Pulse dashboard as the forward-looking financial overview and renaming Intelligence to Statements for clarity.

**What's New:**
- Added new Pulse page as the hero dashboard for forward-looking financial insights
- Renamed Intelligence page to Statements for better clarity
- Updated navigation structure in sidebar and mobile tab bar
- Enhanced page routing and handler organization

**Why This Matters:**
The Pulse dashboard provides users with a forward-looking view of their finances, showing what's coming up and what they can spend. Renaming Intelligence to Statements makes it clearer that this page is for managing financial statements and documents.

**Technical Details:**
- Created `internal/handlers/pulse.go` for new Pulse dashboard
- Renamed `internal/handlers/insights.go` to `internal/handlers/statements.go`
- Updated `internal/views/layouts/base.go` with new navigation structure
- Added Pulse icon to `internal/views/layouts/icons.go`
- Updated route registration in `internal/handlers/handlers.go`
