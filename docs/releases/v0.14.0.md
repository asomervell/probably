## ✨ Features

### Generate Cursor Release Notes for All Releases Including Pre-Releases

Enhanced release automation by integrating Cursor AI to generate comprehensive release notes for all releases, including pre-releases.

**What's New:**
- Generate release notes using Cursor CLI for all releases
- Support for pre-release tags in release notes generation
- Extract semver and release notes into reusable GitHub Actions
- Add feature documentation and release notes templates

**Why This Matters:**
Automated release notes generation ensures consistent, high-quality documentation for every release. This improvement helps maintain clear communication about changes and makes it easier for users to understand what's new in each version.

**Technical Details:**
- Enhanced `.github/workflows/deploy.yml` to use Cursor CLI for release notes
- Created reusable GitHub Actions in `.github/actions/release-notes/`
- Updated release notes generation to handle pre-release tags
- Added comprehensive documentation for the release process
