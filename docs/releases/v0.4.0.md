# Release v0.4.0

**Date:** 2025-12-29  
**Previous Version:** v0.3.5  
**Commit Range:** v0.3.5..v0.4.0

## Changes

### ✨ Features

- serve install.sh from HTTP server (e588646)
- add danger zone settings for account/data deletion (333c059)
- add insights system, merchant search, and enrichment improvements (e3063cd)
- switch to xAI Grok, add fast-path enrichment, and improve localization (33a410a)
- add P2P transfers, institution logos, tag exclusions, and transaction review tracking (74e2d2c)
- implement automatic semantic versioning from commit messages (de7c767)
- add automatic release notes generation from conventional commits (1138ea2)
- improve Firecrawl country hints with transaction description context (e62ec9f)

### 🐛 Bug Fixes

- increase server timeouts for backup import, add install.sh to Docker (29fa11e)
- mobile tab bar styling - use backdrop blur and simplify design (4a332b9)
- Cloud Run deployment and Cloud SQL PG18 compatibility (0d79690)
- improve Teller sync reliability and timestamp tracking (1800096)
- resolve linter errors in shadcn components (fb4084b)
- add --no-cache to docker build to fix cached corrupt layer (fc12a74)
- add goose statement delimiters for DO block in migration 022 (8306936)
- add P2P detection to processing worker and remove old categorize worker (0795a01)
- reject .ico files and remove Bud logo fallback (1de2af0)
- configure git identity before creating semantic version tags (bc00c36)
- correct release notes generation to use previous tag (72ac76a)

### 📝 Documentation

- improve env.example with better organization and comments (5e3fc0f)

### Other Changes

- improve mobile UI and settings consistency (0680b64)
- improve UI consistency and mobile experience (3335abb)
- remove hover quick actions from transaction list (0ff8366)
- Add Bud enrichment integration and merchant management (c0cf85d)
- Shadcn gomponents (8d5f99d)
- Store only filenames in database, construct CDN URLs at display time (35efd45)
- Refactor logo URL handling and cleanup utility scripts (b50124c)
- Update deploy workflow to match GitHub secrets/variables (676776c)
- Add validation for DATABASE_URL secret in deployment workflow (daa29ae)
- Add Cloud SQL instance connection to deployment (042174f)
- Add debug output for DATABASE_URL in deployment (72ef504)
- Fix Teller certificate loading to prefer base64 from env vars over file paths (eaffe8d)
- Improve Teller certificate decoding: trim whitespace and better error messages (0a65f45)
- Suppress directory creation warnings in cloud environments (603457d)
- Remove local storage fallback - always use cloud storage (GCS/S3) (a25a5b6)
- Refactor deploy workflow: separate build and deploy with caching and semver releases (0f30179)
- Ensure logo.dev is always attempted and fix CSS build in GitHub Actions (2c92063)
- Create GitHub releases for all deployments (4e33331)
- Fix gcloud authentication order in workflow (12b77cd)
- Add CSS verification and fix artifact download path (b68c7b8)
- Add verbose logging for CSS inclusion and handle cache warnings (67380ee)
- Fix static file serving to work in Docker containers (120ffcc)
- Removals (c7ce2c2)
