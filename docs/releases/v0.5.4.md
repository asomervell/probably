# Release v0.5.4

## 🎨 UI Improvements

### Remove Hover Quick Actions from Transaction List

Removed hover-triggered quick actions from transaction list for cleaner interface.

**What's Changed:**
- Removed hover-triggered tag dropdown from transaction list
- Removed hover-triggered recategorize button
- Users can still access these features through other means

**Why This Matters:**
Hover actions don't work well on touch devices and can be accidentally triggered. Removing them creates a cleaner, more consistent interface that works better across all devices.

**Technical Details:**
- Removed hover-based interactions
- Simplified transaction list interface
- Better touch device compatibility
