# v1.19.0

**Date:** January 1, 2026
**Previous Version:** v1.18.1

## Summary

This release introduces a powerful spending pace visualization on the Pulse page, enhances AI categorization accuracy with user-defined relationship context, and fixes a critical auto-save bug.

## Features

### Spending Pace Visualization

A new **Spending Pace** line chart on the Pulse page shows your spending trajectory compared to last month:

- **Line Chart Comparison**: Side-by-side visualization of cumulative spending
- **Pace Status Indicators**: Shows if you're spending faster, slower, or on track
- **Full Month Context**: Complete last month trajectory for comparison
- **Largest Expenses**: Top 5 transactions contributing to spending
- **Works Immediately**: No waiting period - uses any available historical data

### Enhanced AI Categorization with My Life Context

The AI categorization system now uses your **My Life** relationships for smarter categorization:

- Relationship context (people, work, assets) included in LLM prompts
- Better categorization of family member transfers
- Improved employer-related transaction identification
- Added "family_member" as a relationship type option

## Bug Fixes

### Auto-Save Entry Preservation

Fixed a critical bug where auto-save was wiping out transaction entries when the amount field was temporarily empty:

- Auto-save only updates entries when amount is valid and non-zero
- Transaction metadata can still be saved independently
- Prevents accidental data loss during editing

## Technical Improvements

- Added more business subtypes for better pattern detection
- Helper functions for one-time vs subscription business classification
- Enhanced transfer matching logic
- Improved pattern detection confidence scoring

## Upgrade Notes

No database migrations required. The spending pace feature works immediately with existing transaction data.
