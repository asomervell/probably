# Release v0.1.24

## ✨ Features

### CI/CD Pipeline for GCP Cloud Run Deployment

Added complete CI/CD pipeline for automated deployment to Google Cloud Run.

**What's New:**
- **GitHub Actions workflow** - Automated build and deployment on push to main
- **Docker image building** - Builds and pushes Docker images to Artifact Registry
- **Cloud Run deployment** - Automatically deploys to Cloud Run service
- **Deployment documentation** - Added comprehensive deployment setup guide

**Why This Matters:**
Automated deployments eliminate manual steps and reduce deployment errors. Every push to main now automatically builds and deploys the latest version.

**Technical Details:**
- Created `.github/workflows/deploy.yml` workflow
- Builds Docker image and pushes to Artifact Registry
- Deploys to Cloud Run with proper configuration
- Added `.github/DEPLOYMENT.md` with setup instructions
- Requires GCP_PROJECT_ID and GCP_SA_KEY secrets
