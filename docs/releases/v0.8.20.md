# Release v0.8.20

## 🧹 Cleanup

### Remove One-Off Utility Scripts and Commands

Cleaned up the codebase by removing utility scripts and one-off commands that are no longer needed.

**What Was Removed:**
- **Logo URL utilities** - Removed `fix_logo_urls.go` and `redownload_logos.go` commands (323 lines)
  - These were one-time migration utilities used to fix logo URL formats in the database
  - Logo management is now handled through the main application
- **GitHub Actions setup scripts** - Removed 5 shell scripts (481 lines total):
  - `check-github-vars.sh` - Verification script for GitHub repository variables
  - `set-github-secrets.sh` - Script for setting GitHub secrets
  - `set-missing-github-vars.sh` - Script for setting missing variables
  - `verify-github-vars.sh` - Verification script
  - `fix-logo-urls.sql` - One-time SQL migration script

**Why This Matters:**
Removing one-off utilities and migration scripts keeps the codebase clean and maintainable. These scripts served their purpose during initial setup and migrations, but are no longer needed in the active codebase. The functionality they provided has either been completed or integrated into the main application.

**Technical Details:**
- Removed 7 files totaling 865 lines of code
- All removed scripts were one-time utilities or setup helpers
- Logo management is now handled through the enrichment system
- GitHub Actions configuration is now managed through the workflow files and repository settings
