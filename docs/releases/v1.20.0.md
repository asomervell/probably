# v1.20.0

**Date:** January 1, 2026
**Previous Version:** v1.19.1

## Summary

This release introduces **AI Chat V2**, a major overhaul of the chat system that brings three powerful new capabilities: vector embeddings for semantic search, persistent chat threads, and real-time streaming of AI reasoning. These features work together to make the AI assistant more intelligent, context-aware, and transparent.

## Features

### Vector Embeddings for Semantic Search

The chat system now uses Google's `text-embedding-004` model to generate 768-dimensional vector embeddings for transactions and entities. This enables semantic similarity search without requiring pgvector extensions.

- **Transaction Similarity Search**: Ask questions like "find transactions similar to Netflix" and the system will use AI embeddings to find semantically similar transactions
- **Cross-Ledger Pattern Detection**: New users benefit from "cold-start" pattern detection that finds similar entities across ledgers
- **Native PostgreSQL Implementation**: Uses PostgreSQL `FLOAT4[]` arrays with custom cosine similarity and Euclidean distance functions

### Persistent Chat Threads

Conversations are now saved to the database, allowing you to continue previous discussions and maintain context across sessions.

- **Thread Management**: Each conversation is stored as a thread with an auto-generated title
- **Thread List Sidebar**: View and navigate between all your previous conversations
- **Message History**: Full conversation history is preserved for follow-up questions
- **HTMX-Powered UI**: Seamless, fast updates without full page reloads

### Streaming LLM Thoughts

Watch the AI think in real-time as it processes your questions. The system now streams the model's reasoning process directly to the chat interface.

- **Real-Time Thought Streaming**: See what the AI is thinking as it analyzes your question
- **Server-Sent Events (SSE)**: Efficient, low-latency streaming of thoughts and results
- **Animated Thinking States**: Visual indicators with animated thought bubbles
- **Vertex AI Integration**: Leverages Vertex AI's streaming API for Gemini models

### UI Improvements

The chat interface has been completely redesigned:

- **Redesigned Chat Interface**: Clean, modern layout with conversation history
- **Thread Sidebar**: Collapsible sidebar showing all conversations
- **Animated States**: Smooth animations for thinking indicators and messages
- **Visual Feedback**: Clear indicators for processing states and operations

## Bug Fixes

- Removed unused non-streaming Vertex AI function
- Fixed duplicate `describeToolCall` function
- Unified `LLMMessage` type across orchestrator strategies

## Database Migrations

Three new migrations are included:

- **051**: Vector embedding columns and similarity functions for entities and transactions
- **052**: Transaction embeddings with legacy `detected_recurring` cleanup
- **053**: `chat_threads` and `chat_messages` tables

## Upgrade Notes

Run database migrations after updating:

```bash
make migrate
```

The embedding service requires either Vertex AI credentials or a Google AI Studio API key configured in your environment.
