# Release v0.8.9

## 🐛 Bug Fixes

### Improved Teller Certificate Decoding

Enhanced Teller certificate loading with better whitespace handling and more informative error messages.

**What Was Fixed:**
- **Whitespace trimming** - Certificates and keys are now trimmed of leading/trailing whitespace before decoding
- **Better error messages** - Error messages now include certificate length to help diagnose issues
- **Improved debugging** - More context in error messages makes it easier to identify certificate configuration problems

**Why This Matters:**
Certificate values from environment variables sometimes include accidental whitespace (newlines, spaces) that break base64 decoding. Trimming fixes this common configuration issue. Better error messages help quickly identify when certificates are malformed.

**Technical Details:**
- Added `strings.TrimSpace()` to certificate and key values before decoding
- Enhanced error messages to include certificate length for debugging
- Improved error context to distinguish between base64 decode failures and file read failures
