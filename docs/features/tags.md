# Tags

Tags are your categorization system for organizing transactions. They form a hierarchical structure that helps you understand where your money goes and where it comes from.

## Hierarchical Structure

Tags can have parent-child relationships:

- **Parent tags**: Broad categories (e.g., "Food & Drink", "Transportation")
- **Child tags**: Specific subcategories (e.g., "Restaurants" under "Food & Drink")
- **Unlimited depth**: Create as many levels as needed

## Features

### Tag Management

- **Create tags** with custom names and colors
- **Edit tags** to update names, colors, or hierarchy
- **Delete tags** (transactions keep their tags, but tag is removed)
- **View tag hierarchy** in a tree structure
- **See usage counts** for each tag

### Default Tags

Get started quickly with pre-built tag categories:

- Income categories (Salary, Freelance, Investment, etc.)
- Expense categories (Food & Drink, Transportation, Shopping, etc.)
- Organized hierarchy
- Customizable after creation

### Tag Colors

- Assign colors to tags for visual organization
- Colors appear in transaction lists and reports
- Helps quickly identify categories at a glance

### Tag Exclusions

Some tags can be excluded from P&L calculations:

- Useful for balance sheet movements (e.g., credit card payments)
- Keeps profit & loss reports focused on income/expenses
- Configurable per tag

### Tag Usage

- See how many transactions use each tag
- Identify unused tags for cleanup
- Understand your spending patterns

## Use Cases

- Organize expenses by category (Food, Transportation, Entertainment)
- Track income sources (Salary, Freelance, Side Hustle)
- Create custom categories for your specific needs
- Build a taxonomy that matches your financial goals
- Generate reports by category

## Best Practices

- Start with broad parent categories
- Add specific child tags as needed
- Use consistent naming conventions
- Review and consolidate similar tags periodically
- Use colors to visually group related categories

## Related Features

- **Transactions**: Tags are applied to transactions
- **Rules**: AI rules automatically assign tags
- **Intelligence**: Tags power category-based reporting
- **Insights**: Tag usage drives financial insights
