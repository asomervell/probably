- **Extractable**: client created only to pass to `NewXxxSyncService` — no direct API calls on the client within the same function.

Result: 11 of the 18 occurrences are extractable.

**Fix:** Added three per-file factory helpers:
- `(*Handlers).newTellerSyncService() (*sync.TellerSyncService, error)` in `teller.go`

- `(*Handlers).newAkahuSyncService() (*sync.AkahuSyncService, error)` in `akahu.go`

Replaced 11 two-step client+service patterns with single helper calls. Dropped now-unused `sync` import from `plaid.go`.
