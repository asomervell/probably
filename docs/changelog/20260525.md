## Reasoning

- `GetAllowedTables`/`GetAllowedColumns` were exported but never called — classic dead API from an incomplete feature. Export-but-never-call is harder to prune later (callers could appear in other binaries), so removing now is the right moment.
- `generateSpendingSuggestions` params were a lie: the signature promised the function would use context, user identity, and time range to generate suggestions, but the body returned a hardcoded slice. Removing them makes the contract honest and eliminates four unused variables the compiler silently ignores.
- Bubble sort was idiomatic of a first-pass implementation. `slices.SortFunc` is shorter, faster in the limit, and what a Go reader expects to see.
